# 📚 Library Management System

A simple yet powerful console application built in **Java** to demonstrate core **Object-Oriented Programming (OOP)** concepts.

## 🚀 Features
- **Add Books:** Register new titles into the inventory.
- **Track Availability:** Check which books are available or borrowed.
- **Console Interface:** Interactive menu for easy navigation.

## 🛠️ Concepts Implemented
- **Encapsulation:** Using private fields and public getters/setters.
- **Collections:** Using `ArrayList` for dynamic data management.
- **Logic Handling:** Real-world state management (Borrowed vs. Available).

## 📖 How to Run
1. Clone the repo.
2. Compile: `javac Main.java`
3. Run: `java Main`
